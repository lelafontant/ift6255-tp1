# IFT6255 - TP1

TREC Evaluation with search engines Lucene, Galago, and Terrier

## Lucene configuration

- Porter stemming
- Krovetz stemming
- Retrieval models
  - BM25: BM25Similarity
  - TFIDF: ClassicSimilarity
  - LMDirichlet: LMDirichletSimilarity
